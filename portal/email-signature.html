How to install (Gmail)

  1. Select the whole block inside the dashed box and copy it.
  2. Gmail → Settings → See all settings → General.
  3. Scroll to Signature, click Create new, name it Cashaw.
  4. Paste. Do not retype anything: pasting carries the formatting.
  5. Under Signature defaults set it for new mail and replies.
  6. Save Changes at the bottom. Send yourself a test.

Rules

  1. Change only the name, the title, and the direct phone. Nothing else.
  2. No quotes, no images, no social icons, no "sent from my iPhone" underneath.
  3. Guest-facing mail from info@kushkations.com does not use this signature. That inbox is KushKations, not Cashaw. See the simple rule.
